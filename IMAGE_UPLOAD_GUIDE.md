## 이미지 업로드 방법

### 방법 1: Windows 탐색기 사용
1. `c:\AI PROJECT BSD\test 1214\byeongjeom-landing\public` 폴더를 엽니다
2. 위 이미지 파일들을 해당 폴더에 복사합니다
3. 파일명이 정확히 일치하는지 확인합니다

### 방법 2: VS Code 사용
1. VS Code에서 `public` 폴더를 엽니다
2. 이미지 파일을 드래그 앤 드롭으로 추가합니다
3. 파일명을 위의 이름으로 변경합니다

## 이미지 최적화 권장사항

### 사무실 외관 사진 (office-exterior.jpg)
- 권장 해상도: 1920x1080px 또는 1600x900px
- 파일 형식: JPG
- 파일 크기: 500KB 이하 (웹 최적화)

### 명함 이미지들 (business-card-*.jpg)
- 권장 해상도: 800x450px 또는 1000x600px
- 파일 형식: JPG
- 파일 크기: 각 200KB 이하

---

## 업로드 후 확인사항

이미지 파일들을 업로드한 후:

1. 개발 서버 재시작:
   ```bash
   npm run dev
   ```

2. 브라우저에서 확인:
   - 회사 소개 섹션에 사무실 사진이 표시되는지 확인
   - 명함 갤러리 섹션에 3개의 명함이 표시되는지 확인
   - 푸터에 사무실 위치 사진이 표시되는지 확인

3. 이미지가 제대로 표시되지 않으면:
   - 파일명이 정확한지 확인
   - 파일 경로가 `public/` 폴더에 있는지 확인
   - 브라우저 캐시를 삭제하고 새로고침

## 현재 적용된 섹션

✅ **AboutSection (회사 소개)**
- 사무실 외관 사진 표시
- 회사 정보 및 핵심 강점 소개

✅ **BusinessCardSection (명함 갤러리)**
- 3개의 명함 이미지 갤러리
- 전화, 이메일, 방문 상담 연락처 카드

✅ **FooterSection (푸터)**
- 사무실 위치 사진 with 찾아오시는 길 정보
- 연락처 및 사업자 정보
